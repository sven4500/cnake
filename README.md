# cnake
Console version of the Snake video game. Written in pure C, straightforward linked list is implemented, adjustable screen size.

```console
+------------------------------------------------------+
|                                                      | 3
|                                                      |
|                                                      |
|                                                      |
|                                                      |
|                                  o                   |
|                                                      |
|                                                      |
|                                                      |
|                                     <.?              |
|                                                      |
|                                                      |
|                                                      |
|                                                      |
+------------------------------------------------------+
```
